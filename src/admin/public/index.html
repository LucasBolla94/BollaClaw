💡 Como funciona a memória

  • Curto prazo: Últimas N mensagens mantidas em SQLite (configurable via MEMORY_WINDOW_SIZE)
  • Longo prazo: Fatos, preferências e instruções extraídos automaticamente e armazenados com embeddings locais (ONNX)
  • Busca inteligente: Memórias só são buscadas quando o contexto precisa — economizando tokens
  • Zero custo de API: Embeddings rodam localmente no CPU, sem chamar nenhuma API externa

Acesso Remoto

Como acessar o painel de qualquer lugar

🖥️ Acesso local (mesma rede)

Se você está na mesma rede do servidor, acesse diretamente:

http://<IP-DO-SERVIDOR>:21086

🌐 Acesso remoto (VPS)

Para acessar seu painel em um servidor VPS de qualquer lugar, siga os passos:

  1. Abra o terminal no seu computador (CMD, PowerShell ou Terminal)
  2. Conecte via SSH com túnel: 
    ssh -L 21086:localhost:21086 usuario@seu-servidor-ip

    Isso cria um túnel seguro. A porta 21086 do servidor será acessível na sua máquina local.

  3. Digite a senha do servidor quando solicitado
  4. Abra o navegador e acesse: 
    http://localhost:21086
  5. Use a senha do painel (gerada na instalação ou alterada por você)

📱 Exemplo completo — Servidor BollaClaw

# 1. Abra o PowerShell / Terminal
# 2. Conecte com túnel SSH:
ssh -L 21086:localhost:21086 ubuntu@server1.bolla.network

# 3. Após conectar, abra no navegador:
#    http://localhost:21086

# 4. Use a senha gerada na instalação para entrar

🔧 Comando rápido (CLI)

Você também pode abrir o painel via terminal no servidor:

bollaclaw web

Mostra o endereço do painel e abre automaticamente se tiver navegador disponível.
