Acesso Remoto

Como acessar o painel de qualquer lugar

🔒 Via SSH Tunnel (Recomendado)

Seguro

O painel escuta apenas em localhost por segurança. Para acessar remotamente, crie um túnel SSH que encaminha a porta do servidor para seu computador.

Abra o terminal no seu computador

Windows: PowerShell · macOS/Linux: Terminal

ssh -L 21086:localhost:21086 ubuntu@51.75.119.144
Digite a senha do servidor

A mesma senha SSH que você usa para acessar o VPS

Abra no navegador

O túnel redireciona a porta local para o servidor

http://localhost:21086
Faça login com a senha do painel

Definida como ADMIN_PASSWORD no .env durante a instalação

ℹ️ Por que SSH Tunnel?

O painel web escuta apenas em 127.0.0.1 (localhost) e não é acessível diretamente pela internet. Isso é intencional — expor painéis de administração na internet pública é um risco de segurança. O túnel SSH criptografa toda a comunicação e usa sua autenticação SSH existente como camada adicional de proteção.
